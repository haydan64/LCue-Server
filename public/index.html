Lcue Server

Version 1.0.0
Created By Haydan Fisher


Notes:

It's suggested to use this software on a private network separate from any POS or other essential systems.
This software can deeply integrate into devices on the network, including Home Assistant, Light boards, Video Devices, and other applicable devices.

It is also VERY recomended that every device on the network have a Static IP address, and/or a unchanging hostname.
